# 这是一个git项目管理工具


在下载很多项目后，但是一直无法管理，只有在想起来的时候，在去搜索项目，然后再去下载更新，很麻烦，所以写了这个工具，一键更新。

项目采用rust完成，二进制文件请从Actions处下载，目前支持windows、mac。
